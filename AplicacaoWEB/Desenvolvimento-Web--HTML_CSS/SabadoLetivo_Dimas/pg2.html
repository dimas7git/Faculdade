Elementos Semânticos gerais

<article> Especifica o conteudo independente e autocontido
<aside> Define algum conteudo além do conteudo em que é colocado
<details> Define detalhes adicionais para o usuario visualizar ou ocultar
<figcaption> Define uma legenda para figure
<figure> Especifica conteudo independente como fotos
<footer> Define um rodapé para o documento
<header> Especifica um cabeçalho para o documento
<main> Especifica o conteudo principal de um documento
<mark> Marca um texto ou destaca
<nav> Navegações em LINKS
<section> Uma seção para o documento
<summary> cabeçalho visível para um elemento
<time> Define data e hora
